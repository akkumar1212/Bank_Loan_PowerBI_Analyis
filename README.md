=> Visuals & Insights
1. Risk Breakdown (Good vs. Bad Loans)

A calculated column was created to classify loans as Good (Fully Paid) or Bad (Charged Off).
Insight: Grades F and G show a noticeably higher default rate, indicating the need for more cautious lending policies for these segments.
